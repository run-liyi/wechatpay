# 🚀 快速开始指南

## 第一步：安装依赖

打开命令行工具（PowerShell 或 CMD），进入项目目录：

```bash
cd d:\program\wechatPay
npm install
```

等待依赖安装完成（需要几分钟时间）。

## 第二步：启动应用

```bash
npm start
```

应用会自动启动并打开窗口。

## 第三步：导入账单

1. 点击左侧 **"📂 选择账单文件"** 按钮
2. 选择项目文件夹中的示例文件：`微信支付账单流水文件(20260513-20260520)_20260520113305.xlsx`
3. 等待解析完成

## 第四步：查看分析结果

使用左侧导航菜单切换不同视图：
- 📊 **数据概览** - 查看收支总览
- 📉 **统计分析** - 查看支付方式和交易类型分布
- 🏷️ **分类汇总** - 按商户/类型查看消费排名
- 📈 **趋势分析** - 查看收支趋势变化
- 📋 **明细查询** - 搜索和筛选具体交易

## 如何获取自己的微信账单？

### 简要步骤：

1. 打开微信 → 我 → 服务 → 钱包 → 账单
2. 点击右上角 `...` → 下载账单
3. ⚠️ **重要**：选择 **"用于个人对账"**（生成Excel格式）
4. 设置时间范围和接收邮箱
5. 输入支付密码验证
6. 24小时内收到邮件，使用微信发送的密码解压

### 详细说明：

- 查看应用内帮助：点击 **"❓ 使用帮助"** 按钮
- 阅读完整文档：`使用指南.md`

## 常见问题

### Q: 安装依赖时出错？
A: 确保已安装 Node.js（14.0或更高版本）

### Q: 无法解析账单文件？
A: 确保选择的是"用于个人对账"导出的Excel文件，不是PDF

### Q: 想打包成可执行文件？
A: 运行 `npm run build:win` 打包Windows版本
