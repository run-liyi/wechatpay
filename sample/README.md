# 脱敏演示数据

本目录下的 `demo-bill.xlsx` 是一份**完全虚构**的微信支付账单样例，字段结构与微信官方导出的账单一致，
但其中的昵称、交易对方、商品、金额、交易单号等**均为人工编造，不含任何真实个人信息**，仅用于：

- 演示与体验本工具的各项分析功能；
- 自动化测试与本地开发的固定输入。

> ⚠️ 请勿将你**真实**的微信账单文件提交到本仓库。仓库已通过 `.gitignore` 忽略根目录下的
> `*.xlsx/*.xls/*.csv`（`sample/` 目录除外），请始终把个人账单放在仓库之外的位置打开。

数据概况：20 笔交易，覆盖收入 / 支出 / 不计收支（中性交易）三类，包含商户消费、红包、转账、
二维码收款、信用卡还款、零钱提现、已退款等典型场景，时间范围 2026-05-13 ~ 2026-05-20。
