# 📊 微信账单分析工具 - 功能演示

## 🎯 核心功能展示

本工具实现了**6个核心功能模块**，每个功能都有完整的实现和精美的可视化界面。

---

## 功能一：数据概览 📊

### 功能描述
快速查看账单的整体情况，包括收支统计、交易汇总和基本信息。

### 展示内容

#### 1. 四大统计卡片
- **💰 总收入**
  - 显示总收入金额（元）
  - 显示收入交易笔数
  - 绿色主题，醒目展示

- **💸 总支出**
  - 显示总支出金额（元）
  - 显示支出交易笔数
  - 红色主题，警示作用

- **📊 净收支**
  - 显示收入-支出的差额
  - 显示收支状态（收入大于支出/支出大于收入）
  - 蓝色主题，平衡展示

- **📝 总交易**
  - 显示交易总笔数
  - 显示账单时间范围
  - 黄色主题，信息展示

#### 2. 收支对比图表
- 使用柱状图直观对比收入和支出
- 绿色柱代表收入，红色柱代表支出
- 鼠标悬停显示具体金额

#### 3. 账单元数据
- 微信昵称
- 账单起止时间
- 导出时间
- 导出类型

### 使用示例
```
示例数据（来自项目中的Excel文件）：
- 总收入：¥133.32（2笔）
- 总支出：¥1511.83（16笔）
- 净收支：-¥1378.51（支出大于收入）
- 总交易：18笔
- 时间范围：2026-05-13 至 2026-05-19
```

## 功能二：统计分析 📉

### 功能描述
深入分析消费习惯和支付偏好，通过多维度统计了解资金流向。

### 展示内容

#### 1. 支付方式分布（环形图）
- 统计各支付方式的使用金额
- 常见支付方式：零钱、零钱通、银行卡、信用卡
- 显示金额和百分比
- 彩色分区，清晰易读

**分析价值**：
- 了解主要使用哪种支付方式
- 评估各支付方式的资金分布
- 优化支付方式选择

#### 2. 交易类型分布（饼图）
- 统计各交易类型的笔数
- 常见类型：商户消费、扫码付款、转账、红包等
- 显示笔数和百分比
- 多彩展示，一目了然

**分析价值**：
- 识别主要消费场景
- 了解交易行为模式
- 发现高频交易类型

#### 3. 交易状态统计（表格）
- 详细列出各状态的统计数据
- 包含：状态名称、交易笔数、总金额、收入、支出
- 常见状态：支付成功、已退款、已转账等

**分析价值**：
- 检查异常交易
- 统计退款情况
- 核对交易完成率

## 功能三：分类汇总 🏷️

### 功能描述
按不同维度对账单数据进行分组统计，找出消费最多的商户或类型。

### 交互控制

#### 1. 分类维度选择
- **按商户**：查看在各个商户的消费情况
- **按交易类型**：统计各类交易的金额
- **按支付方式**：分析各支付方式的使用

#### 2. 排序方式选择
- **按金额**：找出消费金额最高的项目
- **按次数**：找出交易次数最多的项目

#### 3. 显示数量选择
- 前10名：快速查看TOP 10
- 前20名：查看TOP 20
- 前50名：查看TOP 50
- 全部：显示所有分类

### 展示内容

#### 1. 分类排名图表
- 柱状图展示排名
- 数据量少时横向展示
- 数据量多时纵向展示
- 交互式图表，鼠标悬停显示详情

#### 2. 详细数据表格
包含以下列：
- 排名
- 名称（商户/类型/支付方式）
- 交易次数
- 总金额
- 收入金额（绿色显示）
- 支出金额（红色显示）

### 使用场景
```
场景1：找出最常去的商户
- 选择"按商户"
- 选择"按次数"排序
- 查看前10名

场景2：找出花钱最多的类型
- 选择"按交易类型"
- 选择"按金额"排序
- 查看前20名
```

## 功能四：趋势分析 📈

### 功能描述
分析收支随时间的变化趋势，发现消费规律和异常波动。

### 交互控制

#### 1. 时间粒度选择
- **按日**：查看每日收支变化（适合短期分析）
- **按周**：查看每周收支趋势（适合月度分析）
- **按月**：查看每月收支对比（适合长期分析）

#### 2. 数据类型选择
- **收入+支出**：同时显示两条曲线
- **仅收入**：只显示收入趋势
- **仅支出**：只显示支出趋势

### 展示内容

#### 1. 趋势折线图
- 平滑的折线展示收支变化
- 填充区域增强视觉效果
- 绿色曲线代表收入
- 红色曲线代表支出
- 交互式图表，点击查看具体数值

#### 2. 关键统计指标

**日均支出**
- 计算平均每日支出金额
- 评估日常消费水平

**最高单日支出**
- 显示支出最多的一天
- 显示具体日期和金额
- 帮助识别大额消费

**最低单日支出**
- 显示支出最少的一天
- 显示具体日期和金额
- 了解最低消费水平

### 分析价值
- 发现消费高峰期（如月初、月末）
- 识别异常支出
- 评估消费习惯是否稳定
- 制定预算和储蓄计划

## 功能五：明细查询 📋

### 功能描述
查询和筛选具体的交易记录，快速找到特定的交易。

### 交互控制

#### 1. 关键词搜索
- 输入框支持搜索商户名称或商品名称
- 支持模糊匹配
- 按Enter键或点击搜索按钮执行搜索

#### 2. 条件筛选
- **收支类型**：全部 / 收入 / 支出
- **交易状态**：全部 / 支付成功 / 已退款

#### 3. 操作按钮
- **🔍 搜索**：应用筛选条件
- **🔄 重置**：清除所有筛选条件

### 展示内容

#### 1. 筛选统计
- 显示找到的记录数
- 显示筛选结果的总金额

#### 2. 详细交易表格
包含以下列：
- 交易时间
- 交易类型
- 交易对方
- 商品
- 收/支
- 金额（收入绿色，支出红色）
- 支付方式
- 当前状态

#### 3. 表格特性
- 支持滚动查看大量数据
- 鼠标悬停高亮行
- 固定表头，方便浏览

### 使用场景
```
场景1：查找某笔具体交易
- 输入商户名称关键词
- 点击搜索

场景2：查看所有退款记录
- 选择"全部"收支类型
- 选择"已退款"状态
- 点击搜索

场景3：统计某商户的消费
- 输入商户名称
- 选择"支出"类型
- 查看筛选统计
```

## 功能六：报告导出 💾

### 功能描述
将分析结果导出为Excel文件，便于保存、打印或进一步分析。

### 导出内容

生成的Excel文件包含**4个工作表**：

#### 工作表1：汇总统计
- 总收入
- 收入笔数
- 总支出
- 支出笔数
- 净收支
- 总交易笔数
- 账单开始日期
- 账单结束日期

#### 工作表2：分类统计
- 按商户排名的前50名
- 包含：排名、商户名称、交易次数、总金额、收入金额、支出金额

#### 工作表3：每日统计
- 每日的收入
- 每日的收入笔数
- 每日的支出
- 每日的支出笔数
- 每日的净收支

#### 工作表4：明细数据
- 所有交易的完整记录
- 包含所有原始字段

### 使用方法
1. 完成数据分析
2. 点击左侧 **"💾 导出报告"** 按钮
3. 选择保存位置和文件名
4. 点击保存
5. 提示导出成功

### 应用场景
- 长期保存分析结果
- 制作财务报表
- 与他人分享数据
- 在Excel中进行更深入分析

## 🎯 实际应用示例

### 示例1：月度财务分析

**目标**：了解本月的消费情况

**步骤**：
1. 导出本月微信账单
2. 导入到工具
3. 数据概览：查看总收支
4. 分类汇总：找出消费大户
5. 趋势分析：查看每日支出变化
6. 导出报告保存

**收获**：
- 知道本月花了多少钱
- 知道主要花在哪些地方
- 发现异常支出
- 为下月做预算参考

### 示例2：年度消费回顾

**目标**：回顾全年消费情况

**步骤**：
1. 分月导出12个月的账单
2. 在Excel中合并
3. 导入到工具
4. 趋势分析（按月）：查看全年趋势
5. 分类汇总：找出年度消费TOP商户
6. 导出年度报告

**收获**：
- 了解全年消费总额
- 发现消费规律
- 识别可优化的支出
- 制定来年预算

### 示例3：核对退款记录

**目标**：检查所有退款是否到账

**步骤**：
1. 导入账单
2. 明细查询
3. 筛选"已退款"状态
4. 查看退款列表
5. 核对金额和商户

**收获**：
- 确认所有退款
- 发现遗漏的退款
- 及时联系商户

## 💡 使用技巧

### 技巧1：组合使用功能
- 先看概览，再看趋势，最后看明细
- 发现异常时，用明细查询深入调查

### 技巧2：定期分析
- 每月导出一次账单
- 对比不同月份的变化
- 建立消费档案

### 技巧3：关注关键指标
- 日均支出：评估消费水平
- 最大支出项：控制大额开销
- 收支比：判断财务健康

### 技巧4：导出保存
- 定期导出分析报告
- 建立个人财务档案
- 便于长期追踪
