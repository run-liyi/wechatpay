📖 如何获取微信账单数据

导出步骤：

  1. 打开微信，进入 我 → 服务 → 钱包 → 账单
  2. 点击右上角 ... 或 常见问题，选择 下载账单
  3. 选择账单用途为 "用于个人对账"
    • 这样会生成 Excel或CSV格式 的文件
    • （选择"用做证明材料"则会生成带公章的PDF，无法使用）
  4. 设置需要导出的时间范围（最长可导出1年内的数据）
  5. 填写接收邮箱地址
  6. 完成身份验证（输入支付密码）
  7. 系统会在24小时内将加密压缩包发送至邮箱
  8. 解压密码通过微信服务通知发送

数据格式说明：

  • 导出的文件包含：交易时间、交易类型、交易对方、商品、金额、收/支、支付方式、当前状态等字段
  • 文件前16行左右是表头和说明信息，实际数据从第17行开始
  • 本工具会自动识别并解析这些数据

注意事项：

  • 请确保选择"用于个人对账"选项，生成Excel格式文件
  • PDF格式的账单无法被本工具解析
  • 建议定期导出账单进行备份和分析
